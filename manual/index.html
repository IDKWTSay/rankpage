이세돌 뱅온 예약기 매뉴얼

무엇을 해주는 확장프로그램인가요?

우왁굳 및 이세계아이돌 멤버들의 방송 시작을 감지해 자동으로 방송창을 열어주는 확장프로그램입니다. 자동 뱅온 기능 외에도 공지/상태 확인, 편의 기능, 하트포인트 관련 기능을 함께 제공합니다.

작동 전 체크리스트

  • 원하는 방송인의 스위치를 켜 두어야 동작합니다.
  • 최초 설치 시 요청되는 권한 허용이 필요합니다.
  • 방송 시작 후 감지까지 최대 1분 정도 지연될 수 있습니다.
  • 이미 해당 방송에 진입한 상태라면 중복으로 창을 열지 않습니다.
  • 파이어폭스는 자동재생 설정이 필요합니다. 자세한 내용은 FAQ를 확인하세요.

설치 직후 빠른 세팅

  1. STEP 1 크롬 우측 상단 확장 프로그램 메뉴를 엽니다.
  2. STEP 2 목록에서 이세돌 뱅온 예약기를 찾습니다.
  3. STEP 3 핀 고정을 켜 아이콘을 항상 상단에 유지합니다.
  4. STEP 4 아이콘 클릭 후 팝업이 열리는지 확인합니다.

    최초 설치 직후에는 정보가 비어 있을 수 있으며, 약 30분 후 유튭/뱅온 정보가 처음 갱신됩니다.

팝업 UI 미리보기

일반 탭

  • 예약 기능을 키고 끌 수 있는 스위치입니다. 클릭부터 해당 이세돌의 자동 뱅온 기능이 켜집니다.
  • 뱅온정보입니다. 초록색은 뱅온 확정 또는 예정, 노란색은 반반, 빨간색은 후뱅, 회색은 정보가 없는 상태를 뜻합니다.
  • 색이 비어 있으면 정보가 업데이트되지 않았다는 뜻이며, 뱅온정보는 매일 자정에 지워지고 아침에 다시 받아옵니다.
  • 신호 아이콘 위로 마우스를 올리면 툴팁을 확인할 수 있습니다.
  • 스위치를 두 번 누르면 멀티뷰 우선방송 설정이 가능합니다(최대 3인). 세부설정에서 멀티뷰 초과 시 다른 탭 종료로 설정해도 우선방송은 종료되지 않습니다.
  • 오픈방송 숲 아이콘을 클릭하면 해당 채널로 바로 이동할 수 있습니다.
  • 방송이 켜지면 프로필 아이콘 주변에 초록 테두리가 표시됩니다. 클릭하면 해당 방송으로 이동할 수 있고, 마우스를 올리면 썸네일과 방제를 확인할 수 있습니다.
  • 세부설정에서 커스텀 유저 기능을 활성화하면 입력칸이 생성됩니다. 원하는 스트리머 ID를 정확히 입력하고 체크를 누르면 저장되며, 오른쪽 스위치를 켜야 동작합니다.
  • 이세돌, 왁타버스 유튜브를 확인할 수 있습니다. 최신 5개만 표시되며, 새로 올라온 영상은 New 표기가 붙고 30분마다 갱신됩니다.

이모티콘 탭

  • 뱅온예약기 설치 시 채팅창 내 이모티콘 복사/붙여넣기 기능이 항상 활성화됩니다.
  • 사용설명 위로 마우스를 올리면 기본적인 설명이 표시됩니다.
  • 우왁굳+이세돌방에는 응원봉 이모티콘이 기본 할당되어 있으며, `alt`+키 입력 시 4~8개 이모티콘을 빠르게 입력할 수 있습니다.
  • 자동 보내기를 ON으로 두면 단축키 입력 즉시 이모티콘 채팅이 전송됩니다. `alt` 키 차단(기본값) 사용 시 크롬 기준 `alt`+숫자 기능이 제한됩니다.
  • 단시간에 반복 채팅을 치면 채팅 제한을 받을 수 있으니 주의해 주세요.
  • 채팅창에서 먼저 이모티콘을 입력한 뒤 복사하여 원하는 텍스트칸에 붙여넣고 체크를 눌러 저장합니다. 이모티콘이 아닌 요소는 저장되어도 매크로가 동작하지 않습니다.
  • 생방송 진입 후 이모티콘 창을 한 번도 열지 않은 상태라면 기능 동작 시 최초 1회 이모티콘 창이 열릴 수 있습니다.
  • 자동 UP 버튼 기능이 켜져 있으면 동작 충돌이 날 수 있으니, UP 버튼 동작을 확인한 뒤 이모티콘 기능을 사용하는 것을 권장합니다.

OGQ 탭

  • 빈 이미지 칸을 클릭하면 분홍 테두리의 대기 상태가 되며, 한 번 더 누르면 취소됩니다. 대기 상태에서 숲 방송 채팅창 이모티콘을 클릭하면 저장되고 팝업이 닫혀 있어도 상태가 유지됩니다.
  • 텍스트칸에 단축어를 입력한 뒤 오른쪽 체크를 눌러 저장합니다.
  • 채팅창에서 단축어 입력 시 연결된 OGQ가 제안됩니다. ESC로 취소해 일반 채팅으로 보낼 수 있고, ENTER로 OGQ 선택 후 한 번 더 ENTER하면 전송됩니다.
  • 방송 진입 직후 최초 선택 시 이모티콘 창이 잠시 열릴 수 있습니다.
  • 저장된 OGQ 또는 단축어를 바꾸려면 우측 상단 X를 눌러 삭제 후 다시 등록합니다. 이모티콘 탭 오른쪽 영역 클릭으로 OGQ 선택창에 바로 들어갈 수 있습니다.

세부 설정 탭

  • 알람 켜짐 유지를 끄면 뱅온 후 해당 방송인의 스위치가 자동으로 꺼집니다.
  • 플레이어 활성화는 디시인사이드의 숲 플레이어, X(구 트위터), 사운드클라우드, 인스타그램 URL을 embed로 변환해 상호작용할 수 있도록 합니다.
  • 카페공지 포함을 활성화하면 이세돌 최신 공지 5개 표시 시 카페 공지까지 포함하여 최신 5개로 표시됩니다.
  • 하트포인트 기능 ON/OFF를 제어할 수 있으며, 포인트 박스와 시청 외 포인트 수급 관련 동작에 영향을 줍니다.
  • 체크박스로 커스텀 유저 기능을 켤 수 있습니다. 최소 1명에서 최대 4명까지 추가 가능하고, 고정한 유저도 언제든 삭제 후 다른 ID로 변경 가능합니다.
  • 생방송 진입 시 자동으로 UP 버튼을 누르는 기능을 켤 수 있습니다. 진입 후 몇 초 뒤 동작하며, 예약기가 실입장하지 않은 방송에도 적용되고 자정이 넘으면 자동으로 한 번 더 누릅니다.
  • 동시에 4개 이상 생방송이 열릴 때 반응을 미리 정할 수 있습니다. 무시/키지않음/다른 탭 종료 옵션에 따라 창 열기 및 기존 탭 종료 방식이 달라집니다.

하트포인트

하트포인트는 방송 시청과 참여로 모을 수 있는 포인트입니다. 포인트를 모으려는 방송에 들어간 뒤 채팅창 상단 포인트 박스에서 시작 버튼을 눌러야 적립이 시작됩니다.

주기 적립 보너스 하트 시작 버튼 필수

획득 방법 4가지

  1. 방송 시청 중 주기적으로 포인트 획득
  2. 움직이는 하트 클릭 시 보너스 포인트 획득
  3. 채팅 작성 시 포인트 획득
  4. 구독권 선물 시 포인트 획득

구독 상태에 따른 차이

주기 적립 포인트는 비구독자/1티어/2티어 및 구독 개월 수에 따라 차이가 있습니다. 장기 구독 1티어는 신규 2티어보다 더 많은 포인트를 받을 수도 있습니다.

채팅과 보너스 하트는 구독 여부에 관계없이 동일 기준이며, 추후 밸런스 조정이 있을 수 있습니다.

경쟁과 특전

하트포인트는 배팅 기능 대신 누적/7일/30일 랭킹으로 경쟁할 수 있습니다. 10만 포인트 달성 시 뱅온 예약기 특전이 지급되며, 방송인별 왁세돌 특전을 모을 수 있습니다.

리더보드 바로가기

유의 사항

과도하거나 반복적인 채팅은 제재 사유가 될 수 있습니다.

리더보드는 1시간 단위로 업데이트되며, 서버 동기화 시점에 따라 내 점수 반영은 더 늦을 수 있습니다.

ON 옵션은 포인트 박스가 마우스에 반응하지 않을 때만 권장합니다. 최상단 요소로 올라가며 일부 UI와 겹쳐 보일 수 있습니다.

파이어폭스는 자동재생 허용 설정이 필요합니다. 브라우저 설정 > 개인정보 및 보안 > 권한 > 자동 재생에서 모든 웹 사이트 기본값을 허용으로 바꾸거나, 숲 URL만 허용해 주세요.

설정 메뉴를 열지 않아도, 숲 생방송 주소창 왼편의 재생 금지 아이콘에서 해당 URL의 권한을 직접 바꿀 수 있습니다.

일반적으로 그렇지 않습니다. 기본 동작은 1분마다 짧게 수십 KB 수준의 정보만 확인합니다. 생방송 시청 중에는 매크로 사용, UP 버튼 동작 같은 짧은 시점에만 관여하며 화면 정보를 실시간으로 읽고 쓰지 않습니다.

팝업창은 정보량이 많아 일반적인 확장프로그램보다 무거울 수 있지만, 대부분의 환경에서는 사용에 지장을 줄 정도의 랙을 만들지 않습니다.

아래 주소를 열어 `https://play.sooplive.com`의 MEI 점수(`Score`, `Is High`)를 확인해 주세요. `Is High`가 `No`이면 자동재생 제한에 걸린 상태입니다.

새 브라우저/새 PC/쿠키 초기화 이후에는 점수가 다시 낮아질 수 있습니다. 생방송이나 VOD를 여러 번 시청하면 짧으면 하루에서 이틀안에 제한이 해제됩니다.

뱅온 예약기는 1분 주기로 상태를 확인하기 때문에 감지 시점이 최대 1분 이상 지연될 수 있습니다. 뱅온 후 2~3분이 지나도 동작하지 않으면 팝업 설정을 먼저 점검해 주세요.

특히 대상 방송인의 알림 스위치, 동시 방송 3개 이상 켜짐 방지 설정 여부를 확인해 주세요. 설정이 정상인데도 문제가 지속되면 브라우저 재실행 또는 확장프로그램 재설치를 권장합니다.

이미 어떤 방식으로든 한 번 진입했던 방송은 저장되어 중복 진입 방지를 위해 다시 열지 않습니다. 해결되지 않으면 `roman100j12@gmail.com`으로 문의해 주세요.
